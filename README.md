# Операции с графом

Реализация графа на основе списков смежности. Производится добавление начального и конечного узла, топологическая сортировка, построение деревьев доминаторов и постдоминаторов.

## Сборка и запуск

```
cmake -S . -B build
cmake --build build
./build/src/main < <test_name>
```

Результаты визулизируются через graphviz.
